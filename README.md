# Strategy Risk Analysis: Max Drawdown & Volatility
ACC102 Python Data Product | Track 2: GitHub Data Analysis Project

## 1. Project Purpose & Target User
This project calculates and compares key risk metrics for three investment strategies.
It helps users evaluate downside risk and stability for portfolio decision-making.

Intended users: finance students, investors, and analysts.

## 2. Dataset
- Data type: Simulated daily returns
- Period: One year (252 trading days)
- Source: Generated using numpy random normal distribution
- Access date: April 19, 2026

## 3. Python Methods
1. Simulate return data using numpy
2. Calculate Maximum Drawdown (MDD)
3. Calculate Annualized Volatility
4. Visualize results with bar charts
5. Compare risk levels across strategies

Libraries: pandas, numpy, matplotlib

## 4. Key Outputs
- Maximum Drawdown for each strategy
- Annualized Volatility for each strategy
- Two professional comparison charts

## 5. How to Run
